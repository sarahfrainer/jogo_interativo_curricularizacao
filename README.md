# Jogo Interativo sobre Brusque

Projeto desenvolvido para a curricularização da extensão com foco em educação infantil e aprendizado sobre a cidade de Brusque.

## Objetivo

Criar um jogo interativo com perguntas e imagens para ensinar crianças sobre:

* cultura da cidade
* pontos turísticos
* festas típicas
* comidas
* história de Brusque
